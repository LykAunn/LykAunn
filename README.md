### 🏗️ Featured Projects

#### 🦀 [CHIP-8 Emulator](https://github.com/LykAunn/CHIP8-Emulator) | *Rust*

A low-level emulation of the CHIP-8 interpreted programming language.

  * Implemented a full instruction set (Opcodes), timers, and hex keyboard input.
  * Managed CPU cycles and memory mapping using **Rust’s ownership model** to ensure safety.
  * *Key Learning:* Deep understanding of the Fetch-Decode-Execute cycle and bitwise logic.

#### 🧠 [Multi-Level Cache Simulator](https://github.com/inf2c-cs-2022/coursework-2-c-LykAunn) | *C*

A functional simulator of a CPU cache hierarchy (L1/L2) developed for Inf2C-CS at Edinburgh.

  * **Architecture:** Designed a configurable n-way set-associative cache using 3D pointer arrays.
  * **Replacement Logic:** Implemented a global counter-based **LRU (Least Recently Used)** policy.
  * **Memory Safety:** Utilized **Valgrind** to ensure zero memory leaks across complex `malloc`/`free` operations.
  * **Features:** Implemented Write-Back/Write-Allocate policies and sequential "next-line" prefetching.

#### 🎮 [ZooThropic](https://github.com/LykAunn/ZooTrophic) | *Python & Pygame*

A zoo management simulation built from the ground up.

  * Developed a custom game loop and state management system.
  * Engineered efficient sprite-batching and collision detection logic.
  * *Note:* Currently refactoring the core logic into a more modular engine-style architecture.
